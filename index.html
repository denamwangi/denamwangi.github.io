Project: Bubble Reccs web app

A web app that combines the powers of natural language processing using TextRank with data visualization using D3 to synthesize millions of yelp reviews into sets of keywords and short summaries for each business. Users get a snapshot of each business allowing for quicker decisions. Once chosen users may get directions, receive a text with restaurant info using Twilio, or request Lyft cost and time estimates.
